# Obys Agency ï¿½ Landing Page Clone

A front-end practice project that recreates the visual style and interactions of the Obys Agency website. Built with HTML, CSS, and vanilla JavaScript, and enhanced with GSAP, Locomotive Scroll, and Shery.js.

## Features
- Animated loader sequence
- Smooth scrolling with Locomotive Scroll
- GSAP timeline animations and ScrollTrigger integration
- Custom cursor interactions
- Hover image distortion effects (Shery.js)
- Video play/pause interactions
- Responsive layout sections

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- GSAP + ScrollTrigger
- Locomotive Scroll
- Shery.js

## Project Structure
- `index.html`
- `style.css`
- `scrpit.js`
- `assets/`
- `fonts/`

## Getting Started
1. Open `index.html` in a browser.
2. Ensure you have an internet connection (CDN assets are used).
